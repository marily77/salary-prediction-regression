# Salary Prediction with Machine Learning (Regression)

##  Overview

This project focuses on building a regression model to predict employee salaries based on available features.

The goal is to demonstrate a complete machine learning workflow, from data preparation to model training and evaluation.

---

##  Skills Demonstrated

* Data preprocessing
* Feature selection
* Train/test split
* Regression modeling
* Model evaluation

---

##  Dataset

The dataset contains employee-related information such as:

* Age
* Department
* Years since joining
* Other relevant features

The data was cleaned and prepared before applying the machine learning model.

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib / Seaborn

---

##  Workflow

1. Load and prepare dataset
2. Select relevant features
3. Split data into training and testing sets
4. Train regression model
5. Evaluate model performance

---

##  Results

The model is able to predict salary values with reasonable accuracy.

---

##  How to Run

```bash
git clone https://github.com/marily77/salary-prediction-regression.git
cd salary-prediction-regression
pip install -r requirements.txt
```

Open the notebook and run all cells.
